# 🛒 Cadastro de Produtos em Java

Projeto desenvolvido para a disciplina de **Linguagem Orientada a Objetos**, com o objetivo de aplicar conceitos fundamentais da programação orientada a objetos utilizando Java.

---

## 📌 Descrição

Este projeto consiste em um sistema simples de cadastro de produtos, permitindo armazenar e exibir informações como nome e preço. Além disso, o sistema realiza a contagem total de produtos cadastrados por meio de um atributo estático.

---

## 🧠 Conceitos aplicados

* Classes e Objetos
* Construtores (sobrecarga)
* Métodos estáticos
* Atributos estáticos
* Organização básica de projeto Java

---

## 📂 Estrutura do Projeto

```
cadastro-produtos-java/
└── src/
    ├── Produto.java
    └── Principal.java
```

---

## ⚙️ Funcionalidades

* Cadastro de produtos
* Exibição dos dados dos produtos
* Contagem total de produtos cadastrados

---

## ▶️ Como executar

1. Acesse a pasta `src`:

```bash
cd src
```

2. Compile os arquivos:

```bash
javac *.java
```

3. Execute o programa:

```bash
java Principal
```

---

## 🧾 Exemplo de saída

```
Nome: Caderno
Preço: 15.5
----------------------
Nome: Caneta
Preço: 2.5
----------------------
Nome: Mochila
Preço: 120.0
----------------------
Total de produtos cadastrados: 3
```
